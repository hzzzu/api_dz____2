Эта программа использует API NASA,
 чтобы получить фотографии с Марсохода NASA "Curiosity".
 Она запрашивает у пользователя номер сола (количество солнечных дней с момента посадки)
 и название камеры на марсоходе. Затем она отправляет запрос к API NASA с этими данными и
 получает список фотографий, соответствующих введенным параметрам.
  После этого программа выводит количество найденных фотографий и ссылки на них.
